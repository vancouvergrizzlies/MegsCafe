# Meg's Café

A tiny home-based café in Gilbert, Arizona — single-page website.

Hand-pulled lattes, hot or iced, ordered ahead and picked up at the front-porch
counter. Open Mon–Sat, 7am–1pm.

## Files

- `index.html` — the live site (clean single page: **Home · Menu · Order · About**)
- `latteart.jpg`, `logo.png`, `espresso.png` — artwork
- `latteart.original.jpg` — original poster before re-lettering
- `index.riso.html`, `index.backup.html`, `candidates/` — earlier design explorations

## Ordering & scheduling

Ordering, pickup scheduling, and payment are handled by **Hotplate**. The
"Order on Hotplate" button in `index.html` is currently a placeholder
(`https://www.hotplate.com/`) — replace it with the live shop URL once the
Hotplate store is set up.

## Local preview

Open `index.html` in any browser. No build step.
